FunctionUse
vector<int> vecDeclaring a vector
vector<int> vec = {1,2,3}Declaring and initializing the vector at the same time
vector<vector<int>> vDeclaring 2-D vector of int type
vec.size()returns the number of element in the vector
vec.max_size()returns the maximum number of elements vector can hold
vec.capacity()return the size of the storage space currently allocated to the vector expressed as the number of elements
vec.resize(n) resize the current vector so that it contains n element (will delete all the element after the size n)
vec.empty()return 1 if vec is empty else 0
vec.shrink_to_fit()reduce the capacity of vector to fits the size
vec.reserve(n)vector size changed to 100
vec1.swap(vec2)swap all the element between the vector
vec.push_back(5)push value 5 at the end of the vector
vec.insert(vec.begin()+i,n)insert value n at the index i in vector
vec.pop_back()pop the last value from the vector
vec.erase(vec.begin()+i)delete the value stored at the index i in the vector
vec.erase(vec.begin()+i,vec.begin()+j)delete the value stored at the index i to j-1 (here j>i)
vec.clear()delete all the element inside the vector
vec[i]accessing value stored at the index i in the vector
vec.back()return the last element of the vector
sort(vec.begin(),vec.end())sort the vector in ascending order
sort(vec.begin(),vec.end(),greater<int>())sort the vector in descending order
*max_element(vec.begin(),vec.end())return the maximum element from the vec
*min_element(vec.begin(),vec.end())return the minimum element from the vec

Different methods for traversing vector –

  1. Traversing using size – 
for(int i =0; i < vec.size(); i++){
cout<<vec[i]<<" ";
}

2. Traversing vector using iterators –

vector<int> :: iterator v = vec.begin();
while(v!=vec.end()){
cout<<*v<<" ";
v++;
}

3. Traversing using auto variable – 

for(auto i = vec.begin(); i != vec.end(); i++){
cout<<*i<<" ";
}

4. Traversing using auto variable (accessing vector in read-only mode) – 

for(auto i = vec.cbegin(); i != vec.cend(); i++){
cout<<*i<<" ";
}

5. Reverse traversing using auto variable – 

for(auto i = vec.rbegin(); i != vec.rend(); i++){
cout<<*i<<" ";
}

6. Reversing traversing using auto variable (accessing vector in read-only mode)- 

for(auto i = vec.crbegin(); i != vec.crend(); i++){
cout<<*i<<" ";
}

7. Using advanced for loop – 

for(auto i : vec){
cout<<i<<" ";
}
