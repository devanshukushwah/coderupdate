]]> Token in C – https://coderupdate.com//token-in-c/ Thu, 31 Dec 2020 09:43:36 +0000 https://coderupdate.com//?p=461 Read more]]> Token is the smallest unit in a program. Tokens are meaningful to the compiler. In C tokens are divided into 6 categories –

  • Keyword
  • Identifier
  • Constant
  • String
  • Operator
  • Special symbol

Keyword – Keywords are the predefined word in C and have special meaning in the program. C language contains 32 predefined keyword –

Identifier – Identifier refers to the name of variable, function and array. These are the user-defined name and can contain capital alphabet, small alphabet, numbers and underscore( _ ). There are couple of rules programmer should follow while creating any identifier –

  • Identifier must start with either alphabet ( both capital and small ) or underscore ( _ ).
  • Identifier must not start with number.
  • Identifier name should not be keyword.
  • Identifier must not contain any white space
  • Identifier can have any number of capital alphabet, small alphabet, numbers, underscore ( _ ) upto 31 characters.

Constant – Constant refers to the fix value that remains unchanged during the execution of program. ‘const‘ keyword is used to define any identifier as constant.

String – A string is an array of character, containing ‘\0’ at the end of array. The character ‘\0’ ( without ‘ ‘) is known as null character indicates to the compiler that string has ended.

Operator – Operators are the mathematical symbols, used to perform mathematical operation in program. In C language operators are divided into 8 categories –

  • Arithmetic Operator
  • Assignment Operator
  • Bitwise Operator
  • Relational Operator
  • Increment or Decrement Operator
  • Special Operator
  • Logical Operator
  • Conditional Operator

We will learn more about operators in future.

Special Symbol – All the characters other than alphabet and number are special symbol. All of these have special meaning in program.
