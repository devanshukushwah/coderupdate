In c++11, new variant of for loop has been introduced, which iterates over all the elements of a given array, range, collections etc. This variant of for loop can be understand as ‘foreach’ loop.

The structure of for loop is –

for ( var_decl : array ){

statements;

}

Explanation – In the above program, array of data type integer has been initialized –

int arr[] = {5,7,8,9};

Now for loop is initialised with iterator i –

for ( int i : arr)

Loop will run n times, where n is the number of elements in array arr. In each iteration value of i will be the respective value from array arr.
