The data type can be understood as the format of the variable ( or identifier) and what kind of value they are going to hold. In general, four basic data types are widely used –

  • char
  • int
  • float
  • double

We can divide data type into two categories –

Primary / Fundamental /Primitive data type – These data types are predefined in C, and can hold only one value. These data types have different size and characteristics, we will learn more about this here.

Primitive data types are –

Remember that, the size of data type is machine dependent. It may vary according to the machine.

General structure to define primitive data type –

data-type identifier;

Example –

Derived data type – Derived data types are the aggregation of primitive data types. These data types can hold multiple values. In C derived data types are –

  • Array – Array is the collection of similar data items, stored in the contiguous memory location. Values stored in array can be directly accessed using the indicies.
  • Function – A function is a group of statement, used to perform desired task.

Declaring array – The general format of declaring array is

data-type variable-name[size];

here size – how many maximum values can be stored into array.

Example –

Declaring function – The general format of declaring function is –

data-type function-name (parameters);

here parameters means he values passed into the function.

Example –

User-defined data type – User-defined data types are the combination of primitive data types and derived data types. These data type can be used to store the values with the different type but have a relation with each other, ex – name and age of a person can be stored in a single identifier by using user-defined data type.

The operator format is –

identifier = expression1 ? expression2 : expression3;

or in short ? :

Example of ternary operator –

x = ( a > b ) ? a : b;

here x, a, b are the identifier and ‘>’ and ‘?’ are operator.

Ternary operator will written value ‘a’ if condition a> b, otherwise b.

The above syntax can be interpreted as

if ( a> b){

x = a;

}

else{

x = b;

}
